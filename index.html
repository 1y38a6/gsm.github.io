Инструкция по найтройке

Вариант 1

- В разделе меню Вашего девайса «Настройки» выберите
пункт «Сотовая связь».

- Далее войти в подраздел «Параметры данных» и
разрешите «Роуминг данных».

- После перейти в «Сотовая сеть передачи данных».
В этом разделе установить:

- Имя точки доступа, «APN» – globaldata
Имя пользователя – нет

Вариант 2

- Необходимо подключиться к WIFI, открыть браузер
«Сафари» и перейти по ссылке m.apnchanger.org.

- Далее, в ручном режиме выбрать раздел
«Manually», ввести в строку «APN NAME» - globaldata.

- После ввода нажать кнопку
«Установить - Install profile», если потребуется,
необходимо повторно нажать кнопку «Установить -
Install profile».
